# PW1

Projeto desenvolvido utilizando:
- Frontend: HTML, CSS e JavaScript
- Backend: PHP

---

# Como clonar o projeto

Abra o terminal e execute:

```bash
git clone https://github.com/Mrcff/PW1.git
```

Depois entre na pasta:

```bash
cd PW1
```

---

# Como atualizar o projeto

Antes de começar a programar:

```bash
git pull
```

---

# Como enviar alterações

## 1. Verificar arquivos alterados

```bash
git status
```

---

## 2. Adicionar alterações

```bash
git add .
```

---

## 3. Fazer commit

```bash
git commit -m "tipo: descrição"
```

Exemplos:

```bash
git commit -m "feat: adicionar tela inicial"
```

```bash
git commit -m "fix: corrigir erro no login"
```

```bash
git commit -m "style: melhorar layout"
```

---

## 4. Enviar alterações

```bash
git push
```

---

# Tags de commit

| Tag | Descrição |
|---|---|
| feat | Nova funcionalidade |
| fix | Correção de erro |
| style | Alterações visuais |
| refactor | Reorganização do código |
| docs | Documentação |
| test | Testes |

---

# Organização da equipe

- Sempre fazer `git pull` antes de começar
- Não alterar código de outros integrantes sem avisar
- Fazer commits organizados e descritivos
